SBX

El aplicativo SBX es una herramienta para el control de inventarios, como ayuda para el manejo del negocio, SBX es la solución mas practica y económica del mercado, adicional es una aplicación personalizable a la necesidad del cliente.

Modulos

Ventas

Puedes procesar las ventas de una manera rápida,controlada y realizar descuentos.

Compras

Puedes realizar compras y seleccionar el proveedor al cual le estas comprando.

Proveedores

Creación de proveedores, ya sea natural o jurídicos.

Reportes

Usted puede ver los reportes de artículos mas vendidos, ventas, compras, ganacias y perdidas.

Clientes

Creación de clientes, Sistema de Puntos, descuentos y crédito para clientes.

Empleados

Establecer perfiles o permisos para cada empleado de manera individual.
